# Changelog

## 1.0.0 (initial release)

- Complete differential splicing analysis pipeline
- Annotation-free junction discovery (LeafCutter algorithm)
- Multi-way Dirichlet-multinomial GLM with covariate support
- Heterogeneity-aware testing (MAJIQ HET mode)
- MAPQ-weighted junction evidence
- Junction co-occurrence extraction from CIGAR strings
- Effective length normalization (rMATS approach)
- Bootstrap uncertainty quantification (MAJIQ approach)
- Graph-based NMD/PTC functional classification
- Continuous confidence scoring per junction
- Per-event structured diagnostics with confidence tiers
- Cross-tool export (rMATS, LeafCutter, MAJIQ, BED, GTF)
- HTML QC report with embedded figures
- Chromosome-level parallelism
- Parquet-based checkpoint/resume
